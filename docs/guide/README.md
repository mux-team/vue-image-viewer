# 指导

## 安装

### NPM
推荐使用这种方式引入组件
```sh
npm install mux-image-viewer
```

### CDN
您也可以通过引入js文件的方式来使用
```html
<script src="https://unpkg.com/mux-image-viewer@1.0.2/dist/mux-image-viewer.js"></script>
```
